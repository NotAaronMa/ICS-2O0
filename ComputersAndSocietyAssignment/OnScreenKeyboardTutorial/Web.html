Using the micorsoft virtual keyboard

Author: Aaron Ma

The virtual keyboard is a feature present in all versions of Windows newer than windows 8. When activated it appears on the screen as an application which you can use to emulate keyboard input. This feature is ment for the two-in-one laptop form factor to add typing fuctionality to the tablet mode.

In order to access the virtual keyboard you press windows button + u in order to get to the ease of acess settings menu, then you click keyboard and turn on the setting that says "Use the On-Screen Keyboard.
